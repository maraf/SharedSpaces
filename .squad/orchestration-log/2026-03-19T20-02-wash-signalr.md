## Key Decisions

- **accessTokenFactory pattern** — Enables future token refresh without connection rebuild
- **Non-blocking** — SignalR failures don't prevent REST-only operation
- **Post-load timing** — Start after data load to ensure auth ready
- **Full refresh on reconnect** — Simple, consistent strategy; optimizable later
